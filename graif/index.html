graif

概要

zaif とほぼデータ互換の家計簿ソフトです。

操作性については必ずしも Zaif 互換とは限りませんが、 画面のデザイン等多くの部分で Zaif を参考にしています。 (2012/08/19 更新)

動作環境

  • Ruby-GNOME2 (version 1.1.4 で動作確認)
  • Ruby (version 1.9.3p194 で動作確認 1.8 では動作しません)
  • Debian GNU/Linux wheezy で動作確認

スクリーンショット

ダウンロード

sourceforgeからダウンロードできます。

インストール

事前に 

# gem install gtk2
で Ruby-GNOME2 をインストールしておいてください。

Debian/GNU Linux

deb パッケージをダウンロードして dpkg でインストールできます。

ソースからインストール

tarball を展開して 

$ ./configure
$ make
# make install
でインストールできます。

使い方

基本的に Zaif と同じです。

Zaif を使用中の場合 config.xml, data_*.xml を ~/.graif/ にコピーすれば、Zaif と同じように使用できます。

migemo を使うときに migemo の出力する文字コードが環境変数 LANG に依存 しないときはスクリプトの最初の方にある MIGEMO_OUTPUT_UTF8 に false を設 定してください。

Zaif を使用していない場合は、まず口座、分類の設定を行なってください。

Zaif との主な相違点

  • 表示中の月にあるデータを「分類」「口座」「メモ」で検索できます。
  • 集計画面で週間集計を表示する機能はありません。
  • 電卓機能はありません。その代わりに金額の入力欄では数式を使用できます。
  • 残高調整で Zaif は前回調整から計算するようですが、 graif ではその月のデータ (と前の月の繰越データ) から計算しています。
  • Version 0.11 から支出、収入に「特別」属性を指定できるようになりま した。

プラグイン

Version 0.13 から、プラグインで機能を追加できるようにしまし た。プラグインを ~/.graif/plugins にコピーすることで、拡張機能が利用できる ようになります。

save_csv.rb

表示中の月のデータを CSV 形式で保存します。

suica_plugin

Suica に記録されている履歴を graif に読み込みます。

edy_plugin

Edy に記録されている履歴を graif に読み込みます。
