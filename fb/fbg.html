Linux framebuffer 用画像表示プログラム

Download

fbg-0.8.0.tar.gz

提供するもの

  • fbg : framebuffer 用画像ビューア

必要なもの

  • GdkPixbuf, Imlib2 or libstimg 0.1.0
  • TRUE-COLOR の framebuffer を利用できる環境

コンパイル

  • ./configure; make; make install

利用法

  • fbg [-fFcCrR] filename
    -f
    ディスプレイにおさまらない画像を、ディスプレイにおさまる最大の 大きさで表示
    -F
    ディスプレイ全体に拡大/縮小して表示
    -c
    "\033c" を出力して画面を初期化します。
    -C
    frame buffer を 0 クリアした後 "\033c" を出力して画面を初期化し ます。
    -b <background>
    背景色を「RRGGBB」で指定します。
  • 起動中は下記のキー操作が有効です。
    f ディスプレイにおさまる最大の大きさで表示
    F ディスプレイ全体に拡大/縮小して表示
    l 画像を時計回りに90度回転
    r 画像を反時計回りに90度回転
    v 画像を上下反転
    h 画像を左右反転
    o オリジナル画像を表示
    R 再読み込み
    [BS] 前の画像を表示
    [SPACE] 次の画像を表示
    [ENTER] 再表示
    - アニメーションの切り替えを速くします。
    + アニメーションの切り替えを遅くします。
    A 画像縮小時にアンチエイリアスを行うかどうかを切り替えます。
    Q 画像情報などの表示の on, off を切り替えます
    q 終了

制限等

  • framebuffer は 15,16,24,32bpp PACKED-PIXELS TRUE-COLOR 8bpp PACKED-PIXELS PSEUDOCOLOR に対応しています。
  • GIF animation は GdkPixbuf 利用時のみ表示できます。
