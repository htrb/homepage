Linux framebuffer 用画面保存プログラム

Download

fbs-0.3.tar.gz

提供するもの

  • fbs : スクリーンショットを PNG で保存

必要なもの

  • libpng, zlib
  • Linux の framebuffer を利用できる環境

利用法

  • fbs filename [wait]
    filename
    保存するファイル名
    wait
    起動したあと wait 秒後に画面を保存します。

制限等

  • framebuffer は 15,16,24,32bpp PACKED-PIXELS TRUE-COLOR 8bpp PACKED-PIXELS PSEUDOCOLOR に対応しています。

開発環境

  • linux 2.4.18 (Vine Linux 2.5)
  • gcc 2.95.3
  • libpng 1.0.12
  • framebuffer 
         $BN dmesg |grep vesafb
     vesafb: framebuffer at 0xe2000000, mapped to 0xc880d000, size 8192k
     vesafb: mode is 1024x768x16, linelength=2048, pages=4
     vesafb: protected mode interface info at c000:4785
     vesafb: scrolling: redraw
     vesafb: directcolor: size=0:5:6:5, shift=0:11:5:0

その他

  • framebuffer 描画関係のコードは、やまさきしんじさんのサンプルプログ ラムをもとにしています(というかほとんどそのままです)。
  • PNG を出力するコードは imlib2-1.0.6.tar.gz に含まれる loader_png.c を参考にしました。
  • まだ開発途上であり、動作確認も不十分です。使用される際はご自身の責任 でお願いします。
