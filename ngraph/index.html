Ngraph for X11 を GTK+ に移植したものです。使い方は従来と変わりませんが、 個人的な好みで若干の仕様追加を行っています。Windows, macOS でも使用できます。 Update (2025/09/20)

メンテナンスしていないものが多数含まれます。 Update(2005/12/11)

Ruby/GTK による GRA Viewer

Ruby/GTK で GRA Viewer を作ってみました。NGP ファイルの表示も(Ngraph の助 けを借りて) 可能です。 Update(2002/11/12)

Ngraph readline patch

Ngraph for X11 の Ngraph shell に GNU readline ライブラリを組み込んでみま した。 Update(2003/02/20)

pstoedit の GRA 用ドライバ

pstoedit の GRA 用ドライバ を作ってみました。PostScript から GRA への変換が出来ます。 Update(2002/03/06)

gra2ngp

GRA ファイルを NGP ファイルに変換する ruby スクリプトです。めったに使わな いでしょう。 Update(2003/02/12)
