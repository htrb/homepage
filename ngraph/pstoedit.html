pstoedit の GRA 用ドライバ

pstoedit の GRA 用ドライバを作っ てみました。PostScript から GRA への変換が出来ます。

tanaka さんの pstoedit3.32用Ngraph gra形式ドライバーの方がお薦めです。

使用法

  1. pstoedit 3.31 の ソース(pstoedit_3_31.zip)を入手して、適当なディレクトリに展開します。
  2. pstoedit_3.31_drvgra_0.01.patch.gz を先ほど pstoedit を展開したディレクトリにコピーして 
    zcat pstoedit_3.31_drvgra_0.01.patch.gz | patch -p0
    を実行します。
  3. あとは、readme.txt にしたがって 
    cd config
sh configure
cd ../src; make clean; make; make install;
    でコンパイル、インストールできます(必要に応じて su してください)。
  4. 「pstoedit -f gra hoge.ps hoge.gra」で GRA ファイルを生成できます。 詳細は pstoedit のマニュアルを参照してください。

問題点等

  1. 日本語文字列等は閉多角形などで描画されますが、 例えば「田」は塗りつぶされて「■」の様になってしまいます (おそらく "eofill" の解釈の問題だと思います)。
  2. 動作確認が不完全。
